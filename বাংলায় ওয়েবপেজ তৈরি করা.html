আমরা প্রথম অধ্যায়ে শিখব HTML, তারপরে CSS এবং javascript এর ব্যাপারে জানবো । এই ৩ টে জিনিস জানা থাকলে খুব সহজেই যেকোনো ওয়েবপেজ বানানো যাবে।

HTML

HTML এর পুরো কথাটা হল Hyper Text Markup Language. ইন্টারনেট-এর জন্মলগ্নে প্রাথমিক ভাবে বিজ্ঞানী বা গবেষকদের মধ্যে তারের মাধ্যমে বার্তা আদান প্রদানের উদ্দেশ্যে HTML বানানো হয়েছিলো। তাঁদের Computer programming language এর ব্যাপারে সেরকম কোনও ধারণা বা দক্ষতা ছিলোনা । তাঁদের এই দুর্বলতার কথা মাথায় রেখেই ওয়েবপেজ বানাবার জন্য এমন একটি ভাষা উদ্ভাবন করা হয় যা দ্রুত এবং খুবই সহজে শেখা যায়। HTML এর একটি মহান বৈশিষ্ট্য এই যে এতে ভুল বলে কিছু নেই। শিক্ষার্থী যাই লিখুক না কেন, ওয়েবপেজ এ সেটি প্রদর্শিত হবে। তবে এখানে অবশ্যই লিখার কিছু নির্দিষ্ট নিয়ম আছে যেগুলো মেনে লিখলে তবেই ওয়েবপেজ টি নিজের আকাঙ্খিত রূপে প্রকাশ পাবে।

HTML উদ্ভাবনের পর থেকে বহু বছর কেটে গেছে, এবং সময়ের সাথে সাথে এতে বহু পরিবর্তন এসেছে । প্রাথমিক যুগে HTML এর সাহায্যে শুধুই Text based webpage বানানো হতো, পরবর্তী কালে তার সাথে ছবি যোগ হল । আধুনিক কালে ওয়েবপেজ এ প্রায় সমস্ত রকম media ব্যবহার করা হয়ে থাকে। কাজেই বোঝা যায় যে HTML সময়ের সাথে তাল মিলিয়ে আধুনিক হয়ে উঠেছে । বর্তমানে HTML এর যে রূপ টি ব্যবহৃত হয় সেটি হল HTML 5। আমরা এই HTML 5 এর ব্যাপারে এবারে জানবো ।

প্রয়োজনীয় আনুষঙ্গ

HTML-5 এ প্রোগ্রামিং করতে গেলে যা যা লাগে তা প্রায় সমস্ত কম্পিউটার বা আধুনিক স্মার্ট ফোনেই পাওয়া যাবে । এইচটিএমএল লেখবার জন্য প্রয়োজন একটি text based typing software, যেখানে কোনও কথা লিখে save করে রাখা যায়(যেমন notepad, MS word ইত্যাদি), আর একটি internet browsing software, যার মাধ্যমে কোনও ওয়েবপেজ দেখা যেতে পারে(যেমন Google Chrome, Mozilla Firefox, MS Edge ইত্যাদি) ।

লিখবার নিয়ম

HTML-5 লিখবার প্রাথমিক শর্ত হল file টির extension HTML হওয়া বাধ্যতামূলক, যদিও ইদানীং কালে এর আরো কিছু extesion এসেছে, কিন্তু বিশ্বের সর্বত্রই ওয়েবপেজের extension হিসাবে HTML বা HTM ব্যবহার করা হয়। তাই আমরা ওয়েবপেজ বানাতে গেলে প্রথমে একটি text based typing software খুলবো, এবং সেখানে যা কিছু লেখা প্রয়োজন তা লেখা শেষ করার পর file name টির শেষে .html দিয়ে সেভ করব। এর পরে ওই ফাইল টিকে যেকোনো web browser এর সাহায্যে খুললেই আমরা ওয়েবপেজ টিকে যথাযথ ভাবে দেখতে পাব । উল্লেখযোগ্য বিশয় এই যে যদি ওই ফাইল টিতে ভবিষ্যতে কোনও পরিবর্তন করতে হয়, তাহলে আমাদের সেটিকে web browser এর পরিবর্তে text based typing software এর সাহায্যে খুলতে হবে, এবং প্রয়োজনীয় পরিবর্তন করে নেওয়ার পর আবার সেটিকে সেভ করে দিতে হবে।

HTML-5 syntax

HTML-5 শেখার আগে আমরা একটি বিশেষ চিহ্নকে চিনে নেবো, যাকে ট্যাগ বলা হয় । " < " (less tha sign) ">"(greater than sign) এর মধ্যে থাকা কোনও শব্দকে ট্যাগ বলা হয়, যেমন - <html> কে বলা হবে এইচটিএমএল ট্যাগ, <body> কে বলা হবে বডি ট্যাগ ইত্যাদি । ট্যাগ দুই প্রকারের হতে পারে, ওপেনিং ট্যাগ আর ক্লোজিং ট্যাগ; যেমন - <head> হবে ওপেনিং হেড ট্যাগ, আর </head> হবে ক্লোজিং হেড ট্যাগ। ক্লোজিং ট্যাগ এ সব সময় < এর পর একটি /(slash) চিহ্ন আসবে, তারপর ট্যাগ এর নাম টি লিখে > চিহ্ন দিতে হবে । HTML-5 এ ব্যবহৃত অধিকাংশ ট্যাগ এর ই ওপেনিং আর ক্লোজিং ট্যাগ থাকে । আবার কিছু কিছু ট্যাগ আছে জাদের শুধু ওপেনিং হয়, ক্লোজিং হয়না, যেমন -<br> , <hr> ইত্যাদি । আমরা বিভিন্ন ধরনের ট্যাগ এর ব্যাপারে ভবিষ্যতে বিশদে জানবো ।

HTML-5 এ upper case / lower case বা ছোট হাতের লেখা এবং বড় হাতের লেখা নিয়ে কোনও সমস্যা হয়না, হেড ট্যাগ নিম্ন লিখিত সব কটি উপায়েই লেখা যেতে পারে -

  1. <Head>
  2. <HEAD>
  3. <head>
তবে সাধারণত লেখবার সময় ছোট হাতের লেখা ই বেশি ব্যবহার করা হয়ে থাকে । কোনও কারণে কোনও ট্যাগ কে অধিক গুরুত্ব দেওার জন্য বড় হাতের লেখা ব্যবহার করা যেতে পারে, যদিও এরকম ঘটনা খুব ই কম হয়।

HTML-5 এ ওয়েবপেজ লিখতে গেলে সম্পূর্ণ লেখাটিকে এইচটিএমএল ট্যাগ এর মধ্যে লিখতে হয়। যেমন, "নমস্কার!" এই শব্দটি কোনও ওয়েবপেজ এ দেখাতে হলে notepad বা অন্য কোনও typing software এ গিয়ে একটি নতুন ফাইল খুলে লিখতে হবে -
<html>
নমস্কার!
</html>
এর পরে সেটিকে file -> save as -> save as type এ গিয়ে all file types বেছে নিতে হবে। ফাইল এর নাম রাখার সময় খেয়াল রাখতে হবে যেন তার এক্সটেন্সান অবশ্যই .html থাকে । যেমন উপরের ফাইল টির নাম যদি রাখতে হয় First_File, তাহলে file -> save as -> save as type এ গিয়ে all file types বেছে নেওয়ার পরে file name এর অংশে লিখতে হবে First_File.html । এই উপায়ে ফাইল টি সেভ করার পর যেকোনো ওয়েব ব্রাউজার দিয়ে ওই ফাইল টি খুললে দেখা যাবে তাতে "নমস্কার!" লেখাটি ফুটে উঠেছে।
